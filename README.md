# Midnight Bloom Boutique - E-Commerce

### Project Description

A responsive React.js e-commerce frontend application for a luxury perfume boutique. Users can explore seasonal perfume collections for both women and men, featuring smooth interactive UI/UX action buttons.

### Setup Instructions

1. Open the repository on GitHub.
2. Click the green "<> Code" button and select "Download ZIP".
3. Extract the folder on your computer.
4. Run `npm install` and `npm start` to launch the local project environment.
5. View Midnightbloom boutique live: https://midnightbloom-boutique.netlify.app/
